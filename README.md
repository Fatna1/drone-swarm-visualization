# drone-swarm-visualization
Interactive 3D drone swarm visualization dashboard

## Features

- 🚁 **3D Real-time Visualization** - View drone positions, velocities, and states in 3D space
- 🎮 **Interactive Controls** - Modify drone states, battery levels, and positions
- 📊 **Real-time Statistics** - Track modifications and data integrity
- 🎨 **State-based Visualization** - Different colors and symbols for each drone state
- 🔄 **Time Navigation** - Browse through different timepoints

## Drone States

- Taking Off (Orange ●)
- Entering Swarm (Blue ■) 
- Hovering (Cyan ◆)
- Passing By (Purple ✖)
- Attacking (Red ✖)
- Returning (Green ○)
- Parachute Deployment (Yellow □)
- Descending (Red-Orange ◆)
